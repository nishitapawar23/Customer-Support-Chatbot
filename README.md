# Customer Support Chatbot

## Overview

A simple console-based Customer Support Chatbot developed using C++.

The chatbot provides predefined responses to common customer queries such as:

* Greetings
* Order Status
* Refund Requests
* Contact Information

## Features

* Interactive console interface
* String processing
* Conditional logic
* Basic customer query automation
* Object-oriented programming concepts

## Technologies Used

* C++
* Standard Template Library (STL)

## How to Run

1. Compile the program:

```bash
g++ chatbot.cpp -o chatbot
```

2. Run:

```bash
./chatbot
```

## Sample Queries

* hello
* order status
* refund
* contact support
* exit

## Learning Outcomes

* Conditional statements
* String handling
* Problem-solving
* Basic chatbot design
