# Customer Support Chatbot

## Overview

A simple console-based Customer Support Chatbot developed using C++.

The chatbot provides predefined responses to common customer queries such as:

* Greetings
* Order Status
* Refund Requests
* Contact Information

## Features

* Interactive console interface
* String processing
* Conditional logic
* Basic customer query automation
* Object-oriented programming concepts

## Technologies Used

* C++
* Standard Template Library (STL)

## How to Run

1. Open the project in any C++ IDE (Code::Blocks, VS Code, Dev-C++, etc.).
2. Compile the chatbot.cpp file.
3. Run the program.
4. Enter queries such as:
   - hello
   - refund
   - order status
   - contact support
   - exit

## Sample Queries

* hello
* order status
* refund
* contact support
* exit
